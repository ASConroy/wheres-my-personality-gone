# Where's My Personality Gone?

A humorous personal blog and journal about the quiet identity erosion that happens after having kids — and the ongoing, mildly desperate attempt to recover it.

## About

*Where's My Personality Gone?* is a static single-page website styled as an old-school newspaper/journal. It features essays, quizzes, minigames, and a running "missing personality fragments" bulletin tracking everything from spontaneous weekend plans to the ability to stay up past 10pm.

## Features

- Newspaper-style layout with a hero post, sidebar widgets, and posts grid
- Missing Personality Bulletin — a live (ish) inventory of lost traits
- Personality Inventory tracker (Missing / Recovered / Replaced)
- Quizzes, polls, and minigames (coming soon)
- Fully responsive design

## Tech

Pure HTML and CSS. No frameworks. No build tools. No dependencies. Just vibes and mild despair.

Fonts via Google Fonts: [Playfair Display](https://fonts.google.com/specimen/Playfair+Display), [DM Mono](https://fonts.google.com/specimen/DM+Mono), [Lora](https://fonts.google.com/specimen/Lora).

## Running Locally

Open `index.html` in a browser. That's it.
